Data Analytics and Business Intelligence Dashboard

A full-stack Business Intelligence platform built with React, TypeScript, Express.js, and PostgreSQL.

## Features

- JWT Authentication
- KPI Monitoring Dashboard
- Analytics Visualization
- Report Management
- Data Import Module
- PostgreSQL Integration

## Tech Stack

Frontend:
- React
- TypeScript
- Tailwind CSS
- Chart.js

Backend:
- Express.js
- PostgreSQL
- JWT Authentication

## Setup

Frontend:

npm install
npm run dev

Backend:

cd bi-dashboard-backend
npm install
npm run dev
